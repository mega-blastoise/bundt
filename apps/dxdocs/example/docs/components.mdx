# Components

Void Docs includes several built-in components you can use in your MDX files without any imports.

## Callout

Use callouts to highlight important information:

<Callout variant="info">
This is an info callout for general information.
</Callout>

<Callout variant="tip">
This is a tip callout for helpful suggestions.
</Callout>

<Callout variant="warning">
This is a warning callout for things to be careful about.
</Callout>

<Callout variant="error">
This is an error callout for critical warnings.
</Callout>

```mdx
<Callout variant="info">
  Your message here.
</Callout>
```

### Callout Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `variant` | `'info' \| 'tip' \| 'warning' \| 'error'` | `'info'` | Visual style |
| `children` | `ReactNode` | — | Content to display |

## Card

Cards are useful for linking to related content:

<CardGrid>
  <Card title="Getting Started" href="/quickstart">Learn the basics of Void Docs</Card>
  <Card title="Configuration" href="/configuration">Customize your documentation site</Card>
  <Card title="Writing Content" href="/writing-content">MDX authoring guide</Card>
</CardGrid>

```mdx
<CardGrid>
  <Card title="Card Title" href="/link">
    Card description text
  </Card>
</CardGrid>
```

### Card Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `title` | `string` | — | Card heading |
| `icon` | `ReactNode` | — | Optional icon element |
| `href` | `string` | — | Optional link URL |
| `children` | `ReactNode` | — | Card body content |

## CardGrid

Wraps `Card` components in a responsive grid layout.

```mdx
<CardGrid>
  <Card title="One">First card</Card>
  <Card title="Two">Second card</Card>
</CardGrid>
```

## Steps

Display ordered steps with visual indicators:

<Steps>
  <Step title="Install the package">

```bash
bun add void-docs
```

  </Step>
  <Step title="Create your content">

Add MDX files to the `docs/` directory.

  </Step>
  <Step title="Start the dev server">

```bash
bun void-docs dev
```

  </Step>
</Steps>

### Steps Props

| Component | Prop | Type | Description |
|-----------|------|------|-------------|
| `Steps` | `children` | `ReactNode` | `Step` elements |
| `Step` | `title` | `string` | Step heading |
| `Step` | `children` | `ReactNode` | Step content |

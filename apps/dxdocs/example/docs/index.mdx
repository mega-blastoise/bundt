# Welcome to Void Docs

Build beautiful documentation sites with the visual polish of Mintlify and the simplicity of Starlight — powered by React, MDX, and Bun.

## Why Void Docs?

- **Beautiful by default** — Polished, modern theme with automatic light/dark mode
- **Zero runtime overhead** — Pure static HTML output, no framework JavaScript shipped
- **MDX-first** — Write in Markdown with embedded React components
- **Fast builds** — Powered by Bun and esbuild for sub-second builds
- **Simple config** — TypeScript configuration with sensible defaults

## Quick Start

Install void-docs and create your first site:

```bash
bun add void-docs
mkdir docs
echo "# Hello World" > docs/index.mdx
bun void-docs dev
```

# @dxforge/docs

Build documentation sites with the visual polish of premium doc platforms and the simplicity of static generation — powered by React, MDX, and Bun.

**Zero client JavaScript.** Every page is pre-rendered to pure static HTML. No framework runtime ships to your users.

## Why @dxforge/docs?

<CardGrid>
  <Card title="Zero Runtime JS" href="/installation">

  </Card>
  <Card title="MDX-First" href="/writing-content">

  </Card>
  <Card title="Beautiful Defaults" href="/theming">

## Quick Start

```bash
bun add @dxforge/docs
mkdir docs
echo '# Hello World' > docs/index.mdx
bunx dxdocs dev
```

That's it. Open `http://localhost:3000` and start writing.
